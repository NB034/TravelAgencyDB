# База данных TravelAgencyDB

*Создана: С октября на ноябрь 2022 за три недели.*

Проект, созданный в учебных целях. Задумывался как БД для туристического агентства. Скрипт содержит инструкции для создания основных таблиц, архивных таблиц, индексов, пользователей, ролей, представлений, функций, нескольких процедур, нескольких триггеров, и инструкции для помещения в задания для резервного копирования
